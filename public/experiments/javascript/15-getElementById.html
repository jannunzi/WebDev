Get Element By Id

Use method document.getElementById() to retrieve HTML elements by their unique ID. Below there are HTML elements with IDs a, b, and c. 

A = <input id="a"/><br/>
B = <input id="b"/><button onclick="add()">Add</button><br/>
C = <input id="c"/>

The script below gets a hold of the input fields using their IDs and then retrieves their value attribute. The values are strings so we convert them to integers using the parseInt() function. We then add a and b to get the value of c. We use getElementById() again to set the value of input element c. 

<script>
  function add()
  {
    var a = document.getElementById("a").value;
    var b = document.getElementById("b").value;

    a = parseInt(a);
    b = parseInt(b);

    var c = a + b;
    document.getElementById("c").value = c;

    return false;
  }
</script>
