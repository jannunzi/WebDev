Northeastern University CS5610 - Web Development

Course Description

Discusses Web development for sites that are dynamic, data driven, and interactive. Focuses on the software development issues of integrating multiple languages, assorted data technologies, and Web interaction. Considers ASP.NET, C#, HTTP, HTML, CSS, XML, XSLT, JavaScript, AJAX, RSS/Atom, SQL, and Web services. Each student must deploy individually designed Web experiments that illustrate the Web technologies and at least one major integrative Web site project. Students may work in teams with the permission of the instructor. Each student or team must also create extensive documentation of their goals, plans, design decisions, accomplishments, and user guidelines. All source files must be open and be automatically served by a sources server.

Course Expectations

Assignments

There will be six (6) assignments that will help you reinforce the topics discussed in class. Assignments are worth 40% of the grade. Assignments will be submitted through a public code repository available to the instructor and TAs. Assignments must also be hosted online on a public server where they can fully validated. The hosted version must match the lastest version in the source control.

Project

For this course you are expected to create a fully functional Web application on a topic of your choice. There will be six (6) project related assignments that will guide you through the various steps of working on your project. The first assignment will be to discuss and decide the topic and scope of your assignment with your instructor. The project will be hosted on a public server of your choice and fully working by the end of the semester. All the source code for your project must be available to the instructor and TAs through an online repository.

Exams

There will not be any quizes or exams
