Lecture 2: Setting Up Node.js

OVERVIEW: Node.js is an open source, cross platform runtime environment that allows executing JavaScript programs on desktop machines. It is commonly used to create server side networking applications. In this course Node.js will be used to implement HTTP servers that will host HTML content and RESTful Web services.

1. Installing Node.js

Node.js can be installed on most operating systems. Download the installer for your OS from Node.js and follow the installer's instructions. The installer will add several executables to your path of which two will be used extensively throughout the course: node and npm. The node executable is the runtime environment that will allow executing JavaScript programs at the consonle. The npm executable, which stands for Node Package Manager, will allow installing additional Node.js libraries that extend the functionality of Node.js. Once installation is done test whether Node.js has been installed by typing the node command in your terminal. Node.js should give you a prompt for more commands. Type ctrl + c to quit. The following video shows steps on how to download, install, and test a Node.js installation.

Hello.js

Once you have installed Node.js you can try out executing simple JavaScript programs. Create a file called hello.js with the following content 

  console.log("Hello World!");

To execute the program just type node hello.js at your terminal and your program will execute displaying its output in the console. 

  > node hello.js
  Hello World!

2. NPM - Node Package Manager

Like many software development frameworks, they are not terribly interesting on their own. Where they tipically shine is the extensive set of libraries that extend the features of the framework. Node.js has a very active community of developers creating an ever growing collection of open source libraries that have made the framework very popular. These libraries can easily be installed through the use of the npm command or Node Package Manager. Npm downloads packages from a repository and installes them locally on your machine in the form of modules. The repository is hosted at npmjs.com.
