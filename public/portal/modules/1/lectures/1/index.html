Lecture 1: Setting Up Github

OVERVIEW: GitHub is a web-based Git repository hosting service, which offers all of the distributed revision control and source code management (SCM) functionality of Git as well as adding its own features. In this video, we will go through the steps to set up Node.js in your machine.

Steps:

  1. Create account in Github.com.
  2. Download Git client.
  3. Create a test repository in Github.com.
  4. Use terminal to set up local directory.
  5. Link local directory to remote repository.

Common Git Command:

  • git init | Create a new local repository
  • git status | Lists all new or modified files to be committed
  • git add | Add one or more files to staging (index)
  • git push | Send changes to the master branch of your remote repository
  • git commit | Commit changes to head (but not yet to the remote repository)
