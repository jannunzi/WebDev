OVERVIEW

This module deals with concerns of designing and implenting a user interface to interact with the human users of our applications. It covers the two major technologies used to encode the front end:

  • Hypertext Markup Language (HTML) to format plain text for browser rendering
  • Cascading Stylesheet (CSS) to format the look and feel of Web pages

The learning objectives of this modules are

  • Use HTML elements to format Web pages for Web browser rendering.
  • Use CSS rules to format the look and feel of Web pages.
  • Use grid systems to layout Web pages.
  • Use Bootstrap to quickly achieve professional looking and well layout Web pages.

HTML

HTML, or Hypertext Markup Language, is a simple language that uses XML elements to format plain text for Web browsers. HTML uses XML elements to tell Web browsers how to render various pieces of text. For instance the <b> or bold element tells the browser to render a piece of text as bold. The <h1> element is one of several header elements that tells the browser to render a piece of text as a header of some section of the Web page. There are many HTML elements for a variety of formatting options: tables, lists, paragraphs, form elements, and hyperlinks.

CSS

CSS, or Cascading Stylesheets, is a simple language used to format the look and feel of a Web page. Whereas HTML formats the overall structure and logical organization of a Web page, CSS formats how the page looks. CSS formats the font to use, the colors of text, background, and borders, the white spaces between various elements, and page layout.

Box Model

The content in a Web page basically consists of various rentangular regions that can render with the content flow, render vertically or horizontally, render with a specific width, height, border, and white space based on various rules and sizes of the content. Collectively these rules are captured as a "box model". The Box Model lecture covers the various properties that govern the box model.

Bootstrap

The Web development industry has matured enough for design patterns and best practices to emerge. There are enough collective best practices that they can be grouped into coherent frameworks. Several of these frameworks have emerged and become fairly popular amongst the developer community. Twitter's Bootstrap is one of the most popular CSS frameworks that allow developers to quickly style and layout a Web page into consistent, professional looking pages applying industry statndard, best practices.
