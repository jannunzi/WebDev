HTML - Hypertext Markup Language

OVERVIEW: HTML, or Hypertext Markup Language, is a simple language that uses XML elements to format plain text for Web browsers. HTML uses XML elements to tell Web browsers how to render various pieces of text. For instance the <b> or bold element tells the browser to render a piece of text as bold. The <h1> element is one of several header elements that tells the browser to render a piece of text as a header of some section of the Web page. There are many HTML elements for a variety of formatting options: tables, lists, paragraphs, form elements, and hyperlinks.

HTML Semantic Elements

HTML has mostly been concerned with formatting Web pages for human consumption. An increasingly important concern is formatting content consummed by non human, automated processes such as search engines and indexing software. Humans can parse content and identify meaning from the page layout and syntax formatting, e.g., identify dates, first and last names, addresses, main content, etc. To allow automated processes to retrieve the same context information, there has been various efforts to format content with semantic elements.
