Layout

OVERVIEW: A great deal of the concern of displaying content for humans is the how content is layed out. From top to bottom, left to right, columns and rows, indentations and hierarchies - organizing content to effectively convey content is crucial to draw attention to various portions of a Web page. The Layout lecture covers various HTML and CSS techniques to layout Web pages.

Centering Content in HTML

HTML used to provide a center element, but it is no longer supported in HTML 5. The reason is that we now recognize that HTML is not meant for deciding the look and feel of a Web page, but instead it should provide the logical semantic organization of the content in the page and then use a different mechanism, such as CSS, to provide the styling look and feel of the page.

CSS Float



By default block elements such as div and p take up the entire width of the browser window and the height of the content inside it. Inline elements such as a, button and img render with the flow of the surrounding content it is embedded in. The default rendering behavior can be overriden with CSS properties such as float and clear
