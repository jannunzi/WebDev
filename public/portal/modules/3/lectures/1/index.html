JQuery

OVERVIEW: JQuery is one of the most popular JavaScript libraries. JQuery is a free, open-source library that standardizes JavaScript across browser vendors and versions. With jQuery it is easy to query and manipulate the DOM such as retrieving data from a form elements, adding or removing HTML content, hiding or showing different portions of the page, and binding events to complex behaviours.

JQuery also makes it easy to interact with a server using AJAX or Asynchronous JavaScript and XML. Interactions might be sending data requests, receiving data from the server, updating the DOM with data from the server, asynchronously posting data to the server, and sending delete and update commands to the server.

Intro to jQuery


JQuery comes in two main versions: 1.x and 2.x. They both provide the same functionality, but 2.x has dropped support for older browsers such as Internet Explorer 6, 7, and 8. This course uses 1.x

JQuery, and Libraries in general, usually come in an minified and uncompressed version. The minified version is logically equivalent to their uncompressed counterpart. Minified libraries are derived from the uncompressed version by dropping all unnecessary white spaces and renaming variables to smaller equivalent code. The minified version is significantly smaller with the intention of reducing download times. If you open the minified version of a library it will be difficult, if not impossible to make any sense of it. The uncompressed version is the original code with the whitespace and variable names useful for humans to make sense of code. The uncompressed version also helps debugging and stepping through code if you care to debug some elses source code.

The minified version of jQuery can be loaded from a CDN as follows: 

<script src="https://code.jquery.com/jquery-1.11.3.min.js"></script>


Manipulating DOM with jQuery

JQuery makes it easy to programmatically manipulate the DOM. To manipulate the DOM first grab the portion of the DOM to be manipulated using the jQuery or its $ alias. Then the DOM can be added, removed, or changed. The code below retrieves a paragraphs whose ID is intro and adds the text "Hello World !" at the end of this pagraph. 

<p id="example">
  JQuery makes it easy to programmatically
  ...
  at the end of this pagraph.
</p>

var introParagraph = $("#example");
introParagraph.append("Hello World !");

You can change the entire content of an HTML element by using the html function. For instance, the HTML content of the paragraph below was set dynamically with the code that follows it. 

<script>
$("#dynamic").html("This content was generated dynamically");
</script>

JQuery provides a pleathora of functions to dynamically manipulate the DOM. They are all very well documented.


Handling Events with jQuery

JQuery allows binding events to functional behaviours very easily. First grab the DOM element and invoke any of the jQuery event binding methods including:

  • .change() - triggered when the input element changes value
  • .click(), .dblclick() - triggered when the element is clicked or double clicked
  • .focus(), .focusout(), .blur() - triggered when the input field acquires or looses focus
  • .hover() - triggered when the mouse hovers in or out of the element
  • .keydown(), .keypress(), .keyup() - triggered whtn the user presses a key on the keyboard
  • .mousedown(), .mouseenter(), .mouseleave(), .mousemove(), .mouseout(), .mouseover(), .mouseup() - triggered when the user uses the mouse on the element

All the event handling methods are well documented in jQuery's documentation pages


Showing & Hiding HTML with jQuery


Use jQuery's .hide() and .show() methods to programmatically hide and show content. When hiding, the content is not removed from the DOM, it is hidden by setting the element's CSS display property to none. The element is shown by removing the diplay property.
